## Singular-Value-Decomposition
This project was created in Google Colab using Python to explore how Singular Value Decomposition (SVD) works on a small word-frequency dataset.

I used:

• Pandas
• NumPy
• Matplotlib

The project includes:

Creating a word-post matrix
Applying SVD
Reconstructing the matrix with reduced dimensions
Visualizing the results with a heatmap

This helped me better understand dimensionality reduction and matrix approximation in machine learning and data analysis.
